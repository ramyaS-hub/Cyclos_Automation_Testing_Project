# Cyclos Automation Testing Project

## 📌 Project Description
This project demonstrates automation testing of the Cyclos web application using Selenium WebDriver.

## 🛠 Tools & Technologies Used
- Java
- Selenium WebDriver
- TestNG
- Maven

## 📂 Project Structure
- src → Test scripts
- pom.xml → Dependencies
- testng.xml → Test execution configuration

## ✅ Features Automated
- Login functionality
- Navigation between modules
- Basic user operations

## ▶️ How to Run the Project
1. Import project into Eclipse
2. Install Maven dependencies
3. Run testng.xml file

## 📊 Testing Types Covered
- Functional Testing
- Smoke Testing
